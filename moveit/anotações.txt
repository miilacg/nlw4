* Para começar o react com typescript => yarn create react-app nomedoprojeto --template typescript
Dessa forma só tem a funcionalidade do SPA

* Com o Next.js dá pra utilizar SPA, SSR e SSG =>yarn create next-app nomedoprojeto 

* Para inicializar o projeto com o creat react => yarn start

* Para inicializar o projeto com o creat next => yarn dev

* Para instalar o typescript, as tipagens do react, as tipagens do react-DOM e o node => yarn add typescript @types/react @types/react-dom @types/node -D
